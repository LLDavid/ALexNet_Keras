# ALexNet_Keras
This is a very simple implementation of AlexNet on Cifar10. Get all things done with only one command.
1.  prerequisite  
  python 3.5  
  Tensorflow 1.8.0  
  Keras  
  numpy  
2. Implement  
  - For keras model: 
  > python Cifar10_AlexNet_Keras.py.py 
  - For tensorflow model: 
  > Cifar10_AlexNet_TensorFLow.py.py
